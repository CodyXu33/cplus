# C++ 学习大纲

## 1. 学习目标
- 掌握 C++17/20 常用语法与现代编程风格。
- 能独立完成中小型控制台项目。
- 具备阅读开源 C++ 项目和排查问题的能力。

## 2. 学习阶段总览（建议 16 周）
1. 第 1-2 周：环境与基础语法
2. 第 3-5 周：函数、类与面向对象
3. 第 6-8 周：STL 与泛型编程
4. 第 9-11 周：内存管理与现代 C++
5. 第 12-14 周：并发与工程实践
6. 第 15-16 周：综合项目与复盘

## 3. 分阶段内容

### 阶段 A：基础（第 1-2 周）
- 工具链：`g++` / `clang++`、`CMake`、`gdb`
- 语法：变量、条件、循环、数组、字符串、函数
- 输入输出：`iostream`、文件读写
- 练习：
  - 计算器
  - 猜数字游戏
  - 文本文件词频统计（基础版）

### 阶段 B：核心能力（第 3-8 周）
- 类与对象：封装、继承、多态
- 运算符重载、拷贝控制（拷贝构造/赋值）
- STL：`vector`、`map`、`set`、`queue`、`algorithm`
- 模板：函数模板、类模板、基础泛型思维
- 练习：
  - 学生管理系统（类设计）
  - 通讯录（STL 容器）
  - 小型图书管理系统（多文件拆分）

### 阶段 C：进阶（第 9-14 周）
- 内存模型：栈/堆、RAII、智能指针（`unique_ptr`、`shared_ptr`）
- 移动语义：右值引用、`std::move`、完美转发（了解）
- 异常处理、断言、调试技巧
- 并发：`std::thread`、互斥锁、条件变量、原子类型（入门）
- 工程化：CMake 多目录工程、静态库/动态库基础
- 练习：
  - 多线程日志处理器
  - 配置驱动的小型命令行工具

### 阶段 D：项目与提升（第 15-16 周）
- 选择 1 个完整项目：
  - 迷你聊天服务器（TCP）
  - 命令行任务管理器
  - 简易游戏（如贪吃蛇/扫雷）
- 要求：
  - 使用 `CMake` 构建
  - 模块化目录结构
  - 包含 `README` 与测试

## 4. 每周学习节奏（建议）
- 周一到周四：概念学习 + 小练习（每天 1-2 小时）
- 周五：总结与查漏补缺
- 周末：完成一个阶段性小项目

## 5. 推荐实践标准
- 编译参数：`-std=c++20 -Wall -Wextra -Werror`
- 命名建议：
  - 类型名：`PascalCase`
  - 函数/变量：`camelCase`
  - 常量：`kCamelCase`
- 每个模块尽量单一职责，避免超长函数（建议 < 80 行）

## 6. 阶段验收清单
- 能独立使用 STL 解决常见数据处理问题
- 能解释智能指针和 RAII 的意义
- 能用 CMake 组织一个多文件项目
- 能通过调试工具定位常见崩溃问题

## 7. 下一步建议
- 刷题方向：数组、字符串、哈希、二叉树、动态规划
- 阅读开源代码：从小型工具库开始，关注目录结构与接口设计
- 学习方向扩展：网络编程、图形开发、游戏引擎、后端服务
